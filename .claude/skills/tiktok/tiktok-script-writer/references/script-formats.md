# TikTok Script Formats

Five TikTok script formats. Each includes timing structure, on-screen text cues, and spoken lines. Read all scripts aloud before recording — if it sounds scripted, it IS scripted.

---

## Format 1 — Tutorial / How-To

**Length:** 30-60 seconds  
**Use when:** Step-by-step guides, tool demos, workflows

**Key difference from IG Reels tutorial:** Faster pacing. TikTok viewers expect more compression. Cut the setup even shorter. Each step gets less time.

**Structure:**
```
[0:00-0:03] HOOK
  On-screen: "[How to / I automated / X steps to] [specific result]"
  Spoken: "If you [situation], here's exactly what I do:" OR "Here's how I [result] in [time]:"
  Visual: Already doing step 1 — not talking about doing it

[0:03-0:06] WHY (optional, skip if under 30 sec)
  Spoken: "[One sentence on why this matters. Only if non-obvious.]"

[0:06-0:48] STEPS (5-7 sec per step)
  Step 1:
    On-screen: "1. [Step name]" (stays visible 2-3 sec)
    Spoken: "[What you do. What you get. MAX 2 sentences.]"

  Step 2, 3... (same pattern — vary sentence length between steps)

[0:48-0:60] CLOSE + CTA
  On-screen: "Save this" / "Part 2?" / "Comment [word]"
  Spoken: "Save this for when you need it." OR "[Specific CTA]."
```

**Pacing note:** At 130 WPM (natural TikTok pace), 30 seconds = ~65 words. 60 seconds = ~130 words. Write lean.

**Worked example (45 sec):**
```
[0:00-0:03]
On-screen: "Automate dbt testing in 3 commands"
Spoken: "If you're still running dbt tests manually, here's how to stop:"

[0:03-0:15]
On-screen: "1. Install dbt-osmosis"
Spoken: "First, pip install dbt-osmosis. This generates your YAML schema files automatically."

[0:15-0:30]
On-screen: "2. Run dbt-osmosis yaml generate"
Spoken: "One command. It reads your models and writes all your source definitions. 40 minutes of work in 8 seconds."

[0:30-0:42]
On-screen: "3. Add to your CI pipeline"
Spoken: "Add it to your GitHub Actions. Now it runs on every PR. You never write YAML by hand again."

[0:42-0:45]
On-screen: "Save this 🔖"
Spoken: "Save it."
```

## Format 2 — Story

**Length:** 30-90 seconds  
**Use when:** Personal experience, turning points, before/after, lessons from real events

**Key TikTok note:** Start at the END or at the point of maximum tension. Never start at the beginning ("So it all started when..."). TikTok viewers won't wait for the payoff.

**Structure:**
```
[0:00-0:03] HOOK (the ending or the stakes)
  On-screen: "[Where you ended up / the stakes in 6 words]"
  Spoken: "[The result, or the moment of maximum tension]"
  Visual: Show the result/artifact, not the creator standing there talking

[0:03-0:12] SETUP
  Spoken: "[Who you were / what the situation was. Fast. 2-3 sentences max.]"
  On-screen: "[Key context in 3-5 words. Optional.]"

[0:12-0:50] THE STORY (5-8 sec per beat)
  Beat 1: [What happened. Specific. Numbers, dates, names.]
  Beat 2: [The turning point or the moment things changed]
  Beat 3: [The result, cost, or realization]

[0:50-1:10] LESSON (if under 90 sec)
  Spoken: "[One sentence. The thing you'd tell someone in your old situation.]"
  On-screen: "[The key phrase from the lesson]"

[Last 5-8 sec] CLOSE
  "Comment if this hit you." OR end on the strongest line without a CTA.
```

**Worked example (60 sec):**
```
[0:00-0:03]
On-screen: "I lost my biggest client in 48 hours"
Spoken: "January 3rd. $8k/month client. Gone. Here's what I did wrong:"

[0:03-0:12]
Spoken: "I had one client paying 60% of my revenue. Classic mistake. I knew it and ignored it."

[0:12-0:28]
Spoken: "They didn't give me a reason. Just a Slack message saying they were 'going in a different direction.'"
On-screen: "[The Slack message on screen]"

[0:28-0:45]
Spoken: "I spent two weeks panicking. Then I realized — I had never asked them what success looked like. Not once in 6 months."

[0:45-0:55]
Spoken: "Now I send a 'what does success look like?' email on day 1 of every engagement. 0 surprise churns since."

[0:55-1:00]
On-screen: "One question that saves the client."
Spoken: "One question. Send it on day one."
```

## Format 3 — Listicle

**Length:** 15-45 seconds  
**Use when:** Fast-paced N-item tip lists, tools, mistakes, resources

**Key TikTok note:** Much faster than IG. 3-4 seconds per item max. The energy is the format.

**Structure:**
```
[0:00-0:03] HOOK
  On-screen: "[N] [items] [audience] [ignore/miss/need]:"
  Spoken: "[N] [things]. Let's go:"
  Visual: Start on item 1 immediately

[0:03 onwards] ITEMS (3-4 sec per item)
  Each item:
    On-screen: "[Number]. [Item]" (large, fast)
    Spoken: "[1 sentence max. The most surprising or counterintuitive thing about it.]"
    Cut immediately to next item

[Last 3-5 sec] CTA
  On-screen: "Save this" or "Part 2?"
  Spoken: "Save this." (just two words — don't add more)
```

**Rules:**
- On-screen text and spoken line must be DIFFERENT (not the same sentence)
- If they're saying the same thing, cut one
- Most surprising item: second-to-last (not last — viewers are already mentally exiting)
- Under 5 items: too short to be a list. Use a different format.

**Worked example (25 sec):**
```
[0:00-0:03]
On-screen: "5 Python packages nobody talks about:"
Spoken: "5 Python packages I use every day. Starting with:"

[0:03-0:08]
On-screen: "1. Polars" 
Spoken: "Faster than pandas. 10x on large files. Drop-in replacement for most workflows."

[0:08-0:13]
On-screen: "2. Pydantic"
Spoken: "Data validation. If your data isn't the shape you expect, it screams at you before it breaks production."

[0:13-0:18]
On-screen: "3. Rich"
Spoken: "Terminal output that doesn't look like 1992."

[0:18-0:22]
On-screen: "4. Loguru"
Spoken: "Logging without 8 lines of config. One import."

[0:22-0:25]
On-screen: "5. Typer"
Spoken: "Build CLIs in 10 lines. Used it for this whole project."

[0:25]
On-screen: "Save this 🔖"
```

## Format 4 — Opinion / Contrarian

**Length:** 30-60 seconds  
**Use when:** Hot takes, industry pushback, thought leadership

**Key TikTok note:** TikTok rewards genuine opinions over "balanced perspectives." Hedging kills reach. Commit to the claim.

**Structure:**
```
[0:00-0:03] HOOK (the bold claim)
  On-screen: "[Specific contrarian claim in 6 words]"
  Spoken: "[State it directly. No setup. No 'some people think'.]"

[0:03-0:10] STEELMAN
  Spoken: "[One sentence. The strongest version of the opposing view. Be genuinely fair.]"
  (Don't skip this — skipping reads as bad faith)

[0:10-0:40] THE ARGUMENT
  Point 1: [Evidence, experience, or logical step. Specific. 8-10 sec.]
  Point 2: [Second point. Different type of evidence — vary it. 8-10 sec.]
  (Optional) Point 3: [Most surprising piece. 5-8 sec.]

[0:40-0:55] REFRAME
  Spoken: "[New mental model. How they should think about it now.]"
  On-screen: "[The key phrase — 4-6 words]"

[0:55-1:00] CLOSE
  End on the strongest line. No CTA needed for opinion videos.
  "And that's the thing nobody's saying." / "Prove me wrong in the comments."
```

## Format 5 — Behind-the-Scenes / Reveal

**Length:** 20-45 seconds  
**Use when:** Process reveals, "how I actually do X", showing your real setup or workflow

**Key TikTok note:** The reveal must be VISIBLE on screen from second 1. Talking about showing something is less effective than just showing it.

**Structure:**
```
[0:00-0:03] HOOK (show the surprising thing)
  On-screen: "[What they're about to see — the surprising element]"
  Spoken: "[One line that makes the reveal more interesting]"
  Visual: THE REVEAL IS ON SCREEN IN SECOND 1

[0:03-0:15] WALK THROUGH
  Spoken: "[Narrate what they're seeing. Add context they couldn't have.]"
  On-screen: [Labels, arrows, callouts as needed]

[0:15-0:35] WHY IT MATTERS
  Spoken: "[What this means for the viewer's own situation]"
  On-screen: "[Key takeaway phrase]"

[0:35-0:45] CLOSE
  "Comment if you want the full setup." / "Part 2 if this was useful." / End on strongest observation.
```

## Shared rules for all TikTok formats

### Pacing targets
- 130-150 words per minute for educational content
- 100-120 WPM for story content (slower = more emotional weight)
- 160+ WPM for listicles (fast = energy)
- If the script is too long, cut — don't speed up. Rushed speech kills comprehension.

### The read-aloud test
Read every script aloud before using it. If you stumble, the viewer will mentally stumble. Fix every stumble.

### Natural speech additions
Add these sparingly to break the "scripted" feel:
- `[beat]` — natural pause, 0.5-1 sec
- `[here's what surprised me:]` — transition marker
- `Right?` — rhetorical check-in (max 1 per video)
- `Seriously.` — emphasis fragment
- `No joke.` — credibility marker for surprising claims

### TikTok vs YouTube CTA
Never use YouTube-style CTAs on TikTok:
- "Don't forget to like and subscribe!" → wrong platform energy
- "Hit that follow button!" → too desperate

TikTok CTAs that work:
- "Comment [word] if you want part 2."
- "Save this for when you need it."
- "Follow for [specific content type]." (specific > "for more content")
- No CTA — end on the best line (often the strongest choice)
