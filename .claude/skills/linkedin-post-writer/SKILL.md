# LinkedIn Post Writer

Write LinkedIn posts that match [Your Name]'s established voice and audience.

## Process
1. Read references/tone-of-voice.md for writing rules
2. Read references/post-templates.md for structure options
3. Read references/example-posts.md for voice calibration
4. Ask the user for the topic or angle (if not provided)
5. Select the most appropriate template for the topic
6. Write the post following all tone and structure rules
7. Keep posts between 100-200 words unless specified otherwise

## Output Rules
- No emojis
- No hashtags unless requested
- No exclamation marks
- First line must be a standalone hook
- End with a question or single-sentence takeaway
- Use line breaks between paragraphs for readability
- Never start with "I'm excited to" or "Here's the thing"

## Quality Check
Before presenting the post, verify:
- Hook would make someone stop scrolling
- Every sentence adds information (remove filler)
- Tone matches the reference examples
- Length fits within the specified range
