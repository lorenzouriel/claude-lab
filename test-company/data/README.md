# data/ - drop zone

Drop here any file you want CompanyOS to read once: Google Ads export
CSV, sales spreadsheet, contract PDF, meeting transcript, or report
screenshot.

Skills like `/analyze-data` and `/ads-report` read directly from
this folder. You drag the file in, call the command, and receive the
summary.

This is not a final file - it is input. What matters from what is here
becomes an artifact in `_memory/`, `marketing/`, or `outputs/`.

When you have storage MCPs connected (Google Drive, Notion), you can ask
Claude to fetch the files directly from the source - without needing to
download anything here.
