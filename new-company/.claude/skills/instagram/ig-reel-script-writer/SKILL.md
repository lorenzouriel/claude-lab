# IG Reel Script Writer

Write Instagram Reel scripts that hold attention and get shared. Shares to Stories is the #1 signal for Reel reach. Completion rate is #2.

## When to use

- User says "write me a Reel script about X"
- User has a topic and needs a hook + structure for a 15-60 second video
- User wants on-screen text + spoken word cues mapped to timing

## Reel lengths and use cases

| Length | Best for |
|---|---|
| 7-15 sec | Entertainment, quick tips, reactions, "did you know" |
| 15-30 sec | Short tutorials, contrarian takes, story beats |
| 30-60 sec | Educational Reels, step-by-step, storytelling |
| 60-90 sec | Deep dives (reach penalty above 90 sec in most niches) |

Default: 30-60 sec. Ask the user if they have a preference.

## Script structure

```
[0:00-0:03] HOOK
  — On-screen text: [3-6 words, visible before audio loads]
  — Spoken: [What you say in first 3 seconds]
  — Visual: [What's happening on screen]

[0:03-0:10] SETUP / CONTEXT
  — Spoken: [1-2 sentences max. Don't lose them here.]
  — On-screen text: optional reinforcement

[0:10-0:45] BODY
  — Point 1 / Step 1 / Beat 1
    On-screen: [short label]
    Spoken: [The actual content — specific, concrete]
  — Point 2 / 3 / etc. (repeat pattern)

[0:45-0:60] CTA
  — Spoken: [One specific action]
  — On-screen text: [Mirror the spoken CTA]
```

Full format skeletons in `references/script-formats.md`.

## Steps

1. **Gather inputs.** Topic, target length (default 30-60 sec), angle (educational / story / entertaining / contrarian), audience.
2. **Pick format.** Tutorial, story, listicle, breakdown, or opinion. See `references/script-formats.md`.
3. **Write the hook (0:00-0:03).** This is the entire game. Rules:
   - On-screen text must work WITHOUT sound (many people watch muted)
   - Spoken hook must work WITHOUT seeing the screen (many people look away)
   - Both must be interesting alone and even stronger together
   - Do NOT start with: "Hey guys", "So today I wanted to talk about", "Welcome back"
   - DO start mid-action, mid-sentence, or mid-reveal
4. **Write setup (0:03-0:10).** One or two sentences. Don't over-explain. Trust the hook to earn their attention; don't re-earn it.
5. **Write body.** One idea per 5-10 seconds. More specific = more shareable. Vague advice loses viewers mid-Reel.
6. **Write CTA.** One action. Options:
   - "Save this for later" (best for educational)
   - "Share this with [person who needs it]" (best for relatable content)
   - "Comment [word] and I'll DM you [specific thing]"
   - End on the strongest line with no CTA (works for very short Reels)
7. **Humanizer pass.** Read aloud. If it sounds scripted, it IS scripted. Fix it.
8. **Output format.** Timing column + on-screen text column + spoken column. Clear, ready to use on set.

## Hook rules (the most important 3 seconds)

- **Start with a visual action**, not a greeting
- **Text on screen** answers "what is this about" in under 5 words
- **Spoken first line** creates tension or curiosity in under 8 words
- The hook must make someone think: "okay I'll watch this"
- Test: cover the audio. Does the on-screen text still hook you? Cover the screen. Does the spoken line still hook you? Both must be yes.

## Hard rules

- Never start with "Hey guys", "So today", "Welcome back to my page", or any greeting
- No "Don't forget to like and subscribe" (TikTok leak — wrong platform culture)
- The first 3 seconds cannot be a title card or intro animation
- No music-only intros with no on-screen hook
- "Link in bio" must tell them WHAT the link is for, not just that it exists
- Spoken lines must sound natural when read aloud — test by reading, not by reading silently

## Anti-patterns

- Script written as prose that sounds like a blog post read out loud
- Every sentence the same length (spoken word needs rhythm variation)
- Filler phrases: "So basically", "you know what I mean", "at the end of the day"
- Hook that's a question with an obvious answer ("Ever wondered what data engineering is?")
- CTA tacked on as an afterthought after the video's energy has died
