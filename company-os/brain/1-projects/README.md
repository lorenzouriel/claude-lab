# 1-projects/ — active work

Efforts with a clear **goal** and a **deadline**. A client engagement, a
launch, a campaign, a one-off build. Each gets its own folder with a
dedicated `CLAUDE.md` that inherits root context and adds project-specific
rules.

`/new-project` creates folders here (`brain/1-projects/<name>/`).

When a project ships or stalls, move its folder to `brain/4-archive/`.

```
1-projects/
└── acme-rebrand/
    ├── CLAUDE.md        project-specific context
    └── ...              deliverables, notes, assets
```
