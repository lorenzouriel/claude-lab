# 3-resources/ — reference material

Things reused across projects and areas, not tied to any single one:

- `identity/` — brand assets, colors, fonts, logo, `design-guide.md`. Every
  visual task (carousel, slide, landing page) reads this.
- `templates/` — profile templates, skill templates, tool catalog, identity
  examples used by `/install` and `/map-routines`.

Add your own here as the company grows: swipe files, research, SOPs,
reference docs. If a resource becomes irrelevant, move it to
`brain/4-archive/`.
